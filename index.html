ABOUT Me

Fronted Developer

A frontend developer is a type of web developer who focuses on building the user interface (UI) and user experience (UX) aspects of websites or web applications. They are responsible for creating everything that users see and interact with on a webpage. This includes designing layouts, ensuring the website is responsive (works on different devices), and making sure the site is visually appealing and easy to navigate.
